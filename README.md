# Trading Journal

A personal trading journal built with React for recording, analyzing, and reviewing sports betting performance.

Part of the **Zenith Edge** ecosystem — a collection of personal projects focused on building useful tools and improving dev skills.

Unlike a spreadsheet, the focus here is understanding *how* you perform, not just logging wins and losses.

**Live site:** *https://tradinglogs.netlify.app/*
**Repo:** [github.com/AnshMaheshwarii/trading-journal](https://github.com/AnshMaheshwarii/trading-journal)

## Features

- Username + PIN based local authentication
- Portfolio and ROI tracking
- Automatic profit calculations
- Total Return % tracking
- Detailed trade logging
- Strategy performance analytics
- League performance analytics
- Interactive distribution charts
- Monthly performance insights
- Fully responsive design
- Local encrypted storage
- Import and export trading data
- Modern SaaS-inspired UI

## Stack

- React
- JavaScript (ES6+)
- HTML5 + CSS3
- Chart.js
- Local Storage
- Git + GitHub
- Netlify

## Running it locally

```bash
git clone https://github.com/AnshMaheshwarii/trading-journal.git
cd trading-journal
npm install
npm start
```

Open `http://localhost:3000`

## Production build

```bash
npm run build
```

## What this project is about

Built this to have a proper journal that's actually enjoyable to use daily — not just a data dump. Current focus is on better analytics, cleaner usability, and a solid responsive experience. Cloud sync and a few other things might come later as the project evolves.

## About Zenith Edge

Zenith Edge is a concept side project - the idea being to design and build the internal software ecosystem that a real asset management firm could use. Trading dashboards, portfolio tools, reporting systems, analytics platforms — this journal is one piece of that.

It's part hobby, part excuse to build things that are actually complex enough to be interesting.
